# Колядочные Гадания (Kolyadki Fortune Telling)

## Overview
A fortune-telling web application that displays traditional Kolyadki (Christmas/New Year) fortunes.
Users click on an animated card to receive their fortune prediction.

## Tech Stack
- Next.js 16 with TypeScript
- Tailwind CSS for styling
- shadcn/ui for UI components
- framer-motion for animations

## Features

### Main Feature: Fortune Card
- Beautiful animated fortune card with crystal ball icon
- Click to request fortune from API
- Card flip animation when revealing fortune
- Different fortune each time (randomized)
- "Fortune again" button to reset and get new fortune

### UI/UX
- Purple gradient background with animated stars
- Golden header text
- Responsive design (works on mobile and desktop)
- Smooth animations using framer-motion

### API Integration
- POST /api/fortune endpoint
- Supports external AI server via AI_SERVER_URL environment variable
- Falls back to local pre-defined fortunes when AI unavailable
- Returns fortune text in Russian

## Environment Variables
- AI_SERVER_URL (optional): URL for external neural network server for AI-generated fortunes

## Running the Application
```bash
cd app
npm install
npm run dev
```

The application will be available at http://localhost:3000
